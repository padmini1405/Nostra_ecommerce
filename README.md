# Nostra - Modern E-Commerce Platform
- Nostra is a fully responsive, multi-page e-commerce web application. 
- It features a clean user interface and robust client-side functionality for product discovery and customer engagement.

# Features
- Dynamic Collection Page: Includes a real-time search bar and multi-category checkbox filters (Occasion, Color, Arrivals).
- Smart Product Grid: Built with CSS Grid to ensure an even layout that reflows automatically during filtering.
- Interactive Contact Form: Secure form with built-in JavaScript validation and submission alerts.
- Fully Responsive: Optimized for desktop, tablet, and mobile views using Tailwind CSS and Media Queries.
- Navigation: Home, New Arrivals, Most Wanted, Collections, and Contact Us pages.

# Tech Stack
- HTML5: Semantic structure.
- CSS3 & Tailwind CSS: Modern styling and utility-first responsive design.
- JavaScript (Vanilla): DOM manipulation, search algorithms, and form handling.
